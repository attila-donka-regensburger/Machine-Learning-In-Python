Machine Learning strategies notes

  • Numerical: discrete data, continues data
  • Categorical: not measurable values
  • Ordinal: measurable values
  • Mean: is the average value 
    import numpy
            numpy.mean()
  • Median: the middle value 
    import numpy
            numpy.median()
  • Mode: the most common item 
    from scipy import stats
            stats.mode()

Standard Deviation σ

A low standard deviation means that most of the numbers are close to the mean (average) value.

A high standard deviation means that the values are spread out over a wider range.

import numpy
        numpy.std()

Variance σ2

Take the square root of the variance, you get the standard deviation (absolute value).

Variance = multiply the standard deviation by itself

import numpy
        numpy.var()

Percentiles

The value that a given percent of the values are lower than.

import numpy
        numpy.percentile(array, 50)
    #50% is 50, array is array = [...]
